# Expense Tracker Web Application

## Description
This is a simple Expense Tracker web application built using HTML, CSS, and JavaScript.
It helps users track their income and expenses, calculate the current balance, and store data securely using local storage.

## Features
- Intro animation with logo
- User login and logout
- Add income and expense transactions
- Automatic balance calculation
- Transaction history with delete option
- Data persistence using localStorage
- Input validation and error handling
- Clean and responsive UI

## Technologies Used
- HTML
- CSS
- JavaScript
- Browser Local Storage

## How to Use
1. Open `index.html` in a browser.
2. Wait for the intro animation.
3. Click **Login to Start**.
4. Enter username and password.
5. Add income or expense transactions.
6. View transaction history and current balance.

## Data Persistence
All transactions are stored in browser local storage.
Each user has separate stored data.

## Validation
- Empty transaction names are not allowed.
- Amount cannot be zero.
- Error messages are shown for invalid inputs.
